# はまこくExp. 公開用レポジトリ

公共交通オープンデータによって提供される交通データを用いて、横浜国大各所から横浜駅へ向かう際の乗換情報を提供するアプリケーションです。
横浜市営地下鉄・相鉄バス・HELLO CYCLINGなど、大学周辺の主要な交通モーダルに対応しています。

[https://team301.jp/HamakokuEx](https://team301.jp/hamakokuex) にて、実際にアプリケーションが稼働している様子を確認できます

## はまこくExp.の概要

### なぜはまこくExp.を作った？

![国大から横浜駅へ出る際の公共交通モーダル一覧](https://drive.google.com/uc?export=view&id=1TZmplawnlFF5CJybw9tQ66fxIpv8j9SX)

![国大周辺への歩行時間の説明図](https://drive.google.com/uc?export=view&id=1284yobPyNsXh88Nv8pYsLRnGX-ARghcP)

#### 横浜国大周辺を取り巻く公共交通の問題点

- 大学周辺が**坂だらけ**
- 大学周辺に**お店がほとんどない**
- 大学最寄駅が複数あるが、**どの駅に向かうにしても遠い**
- キャンパスが大きいため、**建物によって最寄りの駅やバス停が異なる**

横浜国立大学はゴルフ場に大学を移設した経緯から、歴史的に周囲への公共交通アクセスが不便な状況にあります。それに加えて、大学周辺に歓楽街がないため、夕食や懇親会、飲み会などなど、どこかに集まってご飯を食べようとすると、横浜駅に出るしかありません...

そこで、[公共交通オープンデータ評議会](https://www.odpt.org/)が公開している公共交通機関のリアルタイムデータを活用し、横浜駅に向かう場合の最適な乗り換え方法を提示するアプリケーションを制作し、大学の立地に悩む横浜国大生を救うことにしました！

---

### はまこくExp.は何ができる？

- 横浜国立大学から横浜駅への最適なルートを探索できる
- 大学各所から横浜駅へのおおよその到着予想時間が見れる
- 大学周辺に存在する様々な公共交通の情報を一括で参照できる

使いたいタイミングにすぐ利用できるよう、出発時間を入力するだけでなく、現在時刻から何分後に出発するかを入力するだけでも検索できます！

シェアサイクルだけ、バスだけに絞った検索も可能！

#### アプリケーションが対応している公共交通機関(2026/5時点)

- 鉄道
  - 横浜市営地下鉄
    - 最寄：三ツ沢上町駅
  - 相鉄線
    - 最寄：和田町駅、上星川駅
- バス
  - 横浜市営バス
    - 最寄：国大中央、岡沢町、横浜新道など
  - 相鉄バス
    - 最寄：国大中央、岡沢町、ひじりヶ丘など
  - 神奈中バス
    - 最寄：横浜新道
- シェアサイクル(HELLO CYCLE ドコモバイクシェア)
  - 大学周辺の対応ポート数：１箇所(順次対応予定)
  - 横浜駅周辺の対応ポート数：3箇所
  - ほか、三ツ沢上町駅・和田町駅で鉄道に乗り換える場合にも対応
- 徒歩

### アプリの使い方

![検索前画面](https://drive.google.com/uc?export=view&id=1zILQXbwAIqkmMgMdz9s0UTxMxFtj0IjS)

1. (必要に応じて)出発時間を調整する
2. 出発地をボタンで選ぶ
3. (検索を絞りたい場合のみ)利用したい交通手段を選ぶ
4. 歩く速さを選ぶ(ふつうがおすすめです)
5. 最短ルートを表示！

#### アプリを利用する際の注意点

- **情報の正確性は保証しません**
  - 現在開発中のため、表示される情報には誤りがある場合があります。あくまで参考程度にご利用ください。

## 本プログラムの概要

### 技術スタック

- Runtime: Node.js ("type": "module" によるES Modulesを採用)
- Framework: Express v5.x (最新のExpress 5系を使用)
- Template Engine: EJS v4.x (サーバーサイドレンダリング用)
- 主なミドルウェア・ライブラリ:
  - express-validator: リクエストデータの検証・バリデーション
  - helmet: HTTPヘッダーのセキュリティ対策
  - cors: クロスドメイン通信（CORS）の許可設定
  - morgan: HTTPリクエストのログ出力
  - dotenv: .env ファイルによる環境変数の管理

---

### セットアップ手順

ローカル環境で本プロジェクトを起動するための手順です。

#### 1. リポジトリのクローン

```bash
git clone [https://github.com/team301-yokohama/HamakokuExp-public] [あなたのユーザー名]/[リポジトリ名].git
cd [リポジトリ名]
```

#### 2. 外部ライブラリ等のインストール

```bash
cd app_v1
npm install
```

#### 3. 環境変数の設定

ルートディレクトリにある .env.example ファイルを参考にして、opdtのAPIキーを入力してください

#### 4. アプリケーションの起動

```bash
npm start
```

### アプリケーションを利用する際の注意点

- 相鉄線と神奈中バスのデータは公共交通オープンデータにて現在(2026年5月)公開されていません。データ配信元の規約により、データの再配布ができないため、本レポジトリのコードのみではそれらの検索ができません。
